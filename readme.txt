Otterwp WooCommerce Plugin is fast, fully customizable & beautiful WordPress plugin suitable for WooCommerce storefront.

== Description ==
Otterwp Woocomerces Plugin for WordPress to bring your WooCommerce maximize shopping experience on mobile devices. Otterwp WooCommerce Plugin for WordPress also brings a fully customizable shopping cart.



= Requirements =
* WordPress >= 5.0
* PHP >= 5.6
* Tested up to: 5.8

== Features ==

* Works with WooCommerce plugin shop pages.
* Bings the product information to the customer instead of going to the product page cuts down on wait time.
* fully customizable shopping cart.

= Appearance > Customize =
	- OtterWP Woo Optiions > Cart Options: Choose your cart background color, text color.
	- OtterWP Woo Optiions > Cart Item Options: Choose your cart items background color, text color and variouse other options.
	- OtterWP Woo Optiions > Cart Icon Options: Choose your icon, icon background color, text options, item count and variouse other options.
	- OtterWP Woo Optiions > Single Item Options: Adjust header size, background color, text color and variouse other options.
	

== Installation ==

1. Upload the `otterwp-woo.zip` directory into the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Add the license key you get from freemius.
4. The plugin will automatically take affect after activation, the website on mobile devices with window size that is less then 481 pixels.
5. Customize by going to the Wordpress customizer at the OtterWP Woo Options tab.
